Usefulness Score

Composite score (0.0 - 1.0) combining three weighted factors:

40% Usage rate (invocations/day, max 10) 35% Freshness (1 - decay ratio) 25% Depth (nested files accessed)

Usage Rate

Time-normalized invocations per day since the skill was first discovered. Accounts for skill age so new skills aren't penalized.

Usage Decay

Measures decline in recent usage vs lifetime average. 0.0 = steady or growing, 1.0 = completely stopped. Values above 0.5 are flagged red.

decay = 1 - (recent_14d_rate / lifetime_rate)

Grace Period

Number of days after a skill or file is first discovered before it's included in usefulness scoring. Newly installed skills show a "NEW" badge and are excluded from penalties during this window.
